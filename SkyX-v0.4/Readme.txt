This package contents 4 projects:
----------------------------------------------------------
1. SkyX      - SkyX 0.4
2. SkyXDemo1 - SkyX and terrain integration (Only for Ogre 1.7.X)
3. SkyXDemo2 - SkyX presets
4. SkyXDemo3 - SkyX multi-camera demo

Build instructions:
----------------------------------------------------------
1. SkyX:
   Dependencies: Ogre
   Preprocessor definitions: SKYX_LIB = 1

2. Sample projects: (SkyXDemo1, SkyXDemo2, SkyXDemo3)
   Dependencies: Ogre, SkyX, OIS
   Note: Ogre 1.7 is required for SkyXDemo1 since it uses Ogre TSM, which has been dropped in Ogre 1.8
